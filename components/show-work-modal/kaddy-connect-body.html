My best contribution
I was assigned to automate the order upload - a supplier integration.

Suppliers will just send an email to one of our programmed email, and any attachments within that email that's in CSV format will be processed as orders automatically.
The process involves:
1. First, Kaddy Admin will map the supplier's CSV sample order format.
2. Second, Kaddy Admin will then set a source email. This is where the supplier can send email into.
3. Then Kaddy Admin will set the integration as confirmed. After this, supplier for that integration can now start sending email.
4. The recipient email that the supplier can send into needs to be a verified email in AWS SES.
5. Once a supplier starts sending email, AWS SES will be notified and trigger a Lambda function to process the attachments and will be sent to AWS S3.
6. A command console that is set on cron will process the attachments from AWS S3 into orders.
